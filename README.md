# Installing
- İndirin
- Değiştirin `.env.example` to `.env`
- Token Ekleyin `.env` :: `DISCORD_TOKEN=YOUR_BOT_TOKEN`
- Başlatın `npm start`

# YouTube Together
- baştat `a!yttogether <channelid>`
- odaya katılan bot ile keyfini çıkarın

# Default command prefix
`a!`
